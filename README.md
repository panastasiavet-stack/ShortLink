### Разработано приложение с легкой навигацией при помощи ввода цифр  в соответствии с ТЗ. Для проверки создан пользователь с быстрой ссылкой (логин: Ana, пароль: 11)

### Поддерживается:

•	Регистрация и авторизация пользователей

•	Принимает длинные URL и возвращает уникальные короткие ссылки.

•	Обрабатывает лимит переходов и блокировать ссылку после достижения лимита.

•	Удаляет «протухшие» ссылки по истечении времени.

•	Уведомляет пользователя о недоступности ссылки.

•	Управляться через консоль, включая переходы по сокращенным ссылкам.

•	Сохранение всех пользователей и коротких ссылок в файлы "users.json" и "links.json"
